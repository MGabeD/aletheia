This project is going to be a full stack application with the following goal: Use two systems on my local network to run a backend service to process a stream of data from webcam video, screenshare video, and speach data. It will be responding with questions and building a conversation to mimic an interview for the person using the application. It will be using the two machines to process the video to ensure it is a real person - not a filtered person, processing their screenshare to see what they are doing and quizing them based on that, using their speach + this data to ask insightful questions.
